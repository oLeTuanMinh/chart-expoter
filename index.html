Hướng dẫn sử dụng

  1. Nhập thông tin AWS credentials (Access Key ID, Secret Access Key, Region)
  2. Chọn namespace và metric từ dropdown
  3. Nhập dimensions theo định dạng key=value
  4. Cài đặt khoảng thời gian và các tham số khác
  5. Nhấn "Tạo xem trước" để xem cấu hình JSON
  6. Nhấn "Xuất biểu đồ" để tạo biểu đồ
  7. Tải file PNG về máy

Lưu ý quan trọng:

  • Đảm bảo AWS credentials có quyền CloudWatch:GetMetricWidgetImage
  • Khoảng thời gian không nên quá dài để tránh timeout
  • Dimensions phải chính xác theo namespace đã chọn
  • Có thể sử dụng mẫu có sẵn để bắt đầu nhanh
